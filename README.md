# laravel-smarty
A very lightweight integration of Smarty into Laravel that aims to be very customizable

## Installation
In composer.json:
```
    "require": {
        ...
        "noum/smarty": "^1.0.1"
    },
    "repositories": [
        ...
        {
            "type": "vcs",
            "url": "https://github.com/noum/smarty"
        }
    ]
```
``composer update``
